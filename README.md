# 🏪 Tienda Spring Boot – API REST

## 📌 Descripción
API REST desarrollada con Spring Boot para gestionar
una tienda. Conectada a MySQL con Spring Data JPA.

## 🎯 Tecnologías
- Java 17
- Spring Boot 3.2
- Spring Data JPA
- MySQL 8.0
- Lombok

## 📋 Endpoints disponibles

### Categorías
- GET /categorias
- GET /categorias/{id}
- POST /categorias
- PUT /categorias/{id}
- DELETE /categorias/{id}

### Productos
- GET /productos
- GET /productos/{id}
- POST /productos
- PUT /productos/{id}
- DELETE /productos/{id}

### Clientes
- GET /clientes
- GET /clientes/{id}
- POST /clientes
- PUT /clientes/{id}
- DELETE /clientes/{id}

### Pedidos
- GET /pedidos
- GET /pedidos/{id}
- POST /pedidos
- PUT /pedidos/{id}
- DELETE /pedidos/{id}

## 🚀 Cómo ejecutar
1. Configura MySQL y crea la base de datos tienda_final
2. Actualiza application.properties con tus credenciales
3. Ejecuta TiendaSpringApplication.java
4. Accede a http://localhost:8080
